## About Me

- 공공 API, 인증, 배치, 관리자 기능처럼 실제 서비스에 필요한 흐름을 구현하는 데 관심이 있습니다.
- 백엔드는 Spring Boot, JPA, Security 기반으로 안정적인 API 구조를 만드는 것을 공부하고 있습니다.
- 프론트엔드는 React로 사용자가 자연스럽게 기능을 이해하고 사용할 수 있는 화면을 만드는 데 집중합니다.
- 문제가 생기면 로그와 테스트를 기준으로 원인을 좁혀가며 해결하려고 합니다.

## Personal Project

### 다시, 가족

기획부터 화면 구성, 백엔드 API, 데이터 수집, 관리자 기능까지 직접 구현한 개인 풀스택 프로젝트입니다.

유기동물 조회, 보호소 탐색, 입양 신청, 후원, 입양 후기, AI 상담, 관리자 운영 기능을 하나의 서비스 흐름으로 연결했습니다.

| 구분 | 내용 |
| --- | --- |
| Repository | [animal-platform](https://github.com/Ju-000/animal-platform) |
| Backend | Java 21, Spring Boot 3.5, Spring Security, JPA, MySQL |
| Frontend | React, Vite, React Router, CSS |
| External API | data.go.kr, OpenRouter AI, PortOne |
| 주요 구현 | 세션 인증, CSRF, 공공 API 배치 수집, 보호소 지도 탐색, 관리자 대시보드, 이미지 업로드 검증 |
